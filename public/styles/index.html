Users

#Request Method Path Effect

GET SINGLE "/:id" GET SINGLE USER
POST "/" ADD NEW USER
PUT "/:id" EDIT USER
DELETE "/:id" DELETE USER
POST "/register" REGISTER USER
POST "/login" LOGIN USER
POST "/users/verify" VERIFY USER
